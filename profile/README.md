## What OpenStudy is

A dashboard for your eyes and an MCP server for Claude's. Track courses, lectures, deadlines, and study topics in one place — then let Claude (via Code, Desktop, or the iOS app) create lectures, mark things studied, read your course PDFs page-by-page, and plan your week.

**Your Claude subscription does the thinking. OpenStudy gives it memory.**

## What's here

- [`OpenStudy`](https://github.com/openstudy-dev/OpenStudy) — the main repo: FastAPI + MCP server, React 19 frontend, Supabase schema, full install guide, 44 MCP tools.
